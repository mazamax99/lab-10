Приложение демонстрирует обработку, вывод и работу собственных исключений:
     <ul>
         <li>деление на нуль: ArithmeticException: / by zero</li>
         <li>выход за границы массива - ArrayIndexOutOfBoundsException</li>
         <li>генерация исключения со своим сообщением</li>
         <li>форматированный вывод сообщения пользовательского исключения</li>
     </ul>
